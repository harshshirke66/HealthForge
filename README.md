# ⚡ HealthForge

![HealthForge Banner](./public/banner.png)

### **Forging Your Ultimate Self with AI-Powered Intelligence.**

HealthForge is a premium, high-performance health tracking ecosystem designed for those who demand the absolute best from their bodies. Built with a cutting-edge **Neobrutalist** aesthetic and powered by high-speed AI, it’s not just a tracker—it’s your digital unfair advantage.

---

## 🚀 Key Features

-   **🧠 AI Nutrition Intel**: Forget manual logging. Just tell the AI what you ate, and it handles the macro-breakdown with lightning-fast accuracy using the Groq Llama-3 backend.
-   **🌙 Deep Sleep Analytics**: Track your recovery with precision. Visualize sleep cycles and quality through our high-contrast dashboard.
-   **⚡ Live Performance Metrics**: Monitor weight, heart rate, and activity trends with real-time feedback.
-   **🤖 AI Health Assistant**: A dedicated chat interface to answer your complex fitness questions and provide actionable habit coaching.
-   **🛠️ Neobrutalist Design System**: A "bold as hell" UI featuring high-contrast HSL palettes, thick borders, and hard shadows for maximum readability and attitude.
-   **📱 Mobile-First Excellence**: Engineered to be flawlessly responsive. Your data looks stunning on a 30-inch monitor and a 6-inch phone alike.

## 🛠️ Tech Stack

### **Frontend**
-   **Next.js 14** (App Router)
-   **TypeScript** (Type-safe everything)
-   **Vanilla CSS** (Custom Neobrutalist design system)
-   **Lucide Icons** (Sharp, consistent iconography)

### **Intelligence Layer**
-   **Groq Cloud API** (Ultra-low latency AI processing)
-   **Llama-3 70B** (State-of-the-art health logic)

## 🎨 Design Philosophy: Neobrutalism

HealthForge breaks away from the "boring corporate clean" look. We embrace **Neobrutalism**:
-   **Electric Colors**: High-saturation Yellow, Indigo, and Pink.
-   **Bold Outlines**: 3px - 4px black borders on every element.
-   **Hard Shadows**: 6px - 10px shadows for a physical, "off-the-page" feel.
-   **Raw Typography**: Massive, ultra-bold headings that command attention.

## ⚙️ Getting Started

### 1. Clone the Forge
```bash
git clone https://github.com/harshshirke66/HealthForge.git
cd HealthForge
```

### 2. Install Dependencies
```bash
npm install
```

### 3. Environment Secrets
Create a `.env` file in the root and add your Groq key:
```env
GROQ_API_KEY=your_api_key_here
```

### 4. Ignite the Engine
```bash
npm run dev
```
